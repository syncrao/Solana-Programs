# 🚀 Solana Programs Lab

A collection of **Solana smart contracts (programs)** built using the **Anchor framework** to learn and explore Web3 development.

This repository contains multiple mini-projects ranging from beginner to advanced concepts like PDAs, vaults, token programs, and DeFi primitives.

---

* Solana program architecture
* Anchor framework fundamentals
* PDAs (Program Derived Addresses)
* Lamports & account model
* CPI (Cross Program Invocation)
* Token programs (SPL)
* Smart contract security basics

---

## 🧱 Projects Included

### 1. Vault Program

* Deposit SOL
* Withdraw SOL
* PDA-based ownership

### 2. Counter Program

* Basic state management
* Increment/decrement logic

### 3. Token Vault 

* SPL token deposits
* Secure withdrawals

### 4. Multi-user Vault

* Shared pools
* Advanced PDA design
